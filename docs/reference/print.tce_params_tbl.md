# Print method for `tce_params()` output

Print method for
[`tce_params()`](https://monitoramento.sepe.pe.gov.br/tcepe/reference/tce_params.md)
output

## Usage

``` r
# S3 method for class 'tce_params_tbl'
print(x, ...)
```

## Arguments

- x:

  An object returned by
  [`tce_params()`](https://monitoramento.sepe.pe.gov.br/tcepe/reference/tce_params.md).

- ...:

  Passed to the next print method.

## Value

`x`, invisibly.
